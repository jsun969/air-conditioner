# Air Conditioner

> Inspired by [YunYouJun/air-conditioner](https://github.com/YunYouJun/air-conditioner)

## Features

### Pros

- **Wireless remote control**
- Turn on it anytime and anywhere
- Low power consumption
- Easy to use
- Portable

### Cons

- No wind

## Installation

```
pnpm i
```

## Development

```
pnpm run dev
```

## Deployment

```
pnpm run build
pnpm run start
```
